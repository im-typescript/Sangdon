# TypeScript
## How to Install TypeScript With Visual studio
- node.js 가 깔려있다고 생각하고 진행합니다.
```
    npm init 
    npm install typescript 이렇식으로 전역에 깔지 않으면 저장했을 때 컴파일 되지 않는다
    npm install -g typescript
    tsc --version
```

ts-node를 설치하는 이유 :
- tsc타입스크립트 코드를 ES5형식으로 변환시킬 뿐, 실행하지 않기 때문에 ts-node를 설치해서 실행시킬 수 있습니다.

```
    npm i -g ts-node
```


chance,ramda와 같은 라이브러리들이 제공하는 함수들을 올바르게 사용했는지 검증하는 역할을 합니다.
ex) @types/ranmda

Promise를 사용하려면 @types/node를 설치해야됩니다.
